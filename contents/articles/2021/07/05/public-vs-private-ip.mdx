パブリック IP
インターネットから直接アクセスできる IP アドレスのこと

プライベート
個人のパソコンなどは IP アドレスが存在しているが、ルーターがないと外からアクセスはされない。
範囲が決まっている

- 10.0.0.0 - 10.255.255.255 (10.0.0.0/8) 大きい組織で使われている
- 172.16.0.0 - 172.31.255.255 (172.16.0.0/12) AWS でデフォルトで設定されている範囲
- 192.168.0.0 - 192.168.255.255 (192.168.0.0/16) 自宅のパソコンなど、WIFI ルーターなど（？）
