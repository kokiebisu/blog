## 用語

### クライアント

→ ブラウザのこと。実際に URL を打つことによって欲しい情報をインターネットから拾ってきてもらう。欲しいことを伝えることを Request（リクエスト）と呼ぶ。

### サーバー

→ ブラウザからのリクエストを読み取って、その必要に応じてページを表示させたり、データを送り返したり様々な行動をとってくれる。これらの行動を Response（レスポンス）と呼ぶ

## DNS とは？

> **Domain Name Server**の略。ドメイン(google.com や facebook.com など）をみて、それに該当するサーバーの住所を教えてくれる

インターネットとはサーバーの団地のようなもの。ちゃんとそれぞれのサーバーには住所が割り当てられていてるが私たちが認識できる言語の住所ではない。サーバーが置かれている住所のことを一般的に IP アドレスといい、数字に羅列で構成されている。

```
https://216.58.211.206
```

DNS は google.com で登録されているアドレスを本来の IP アドレスに変換してくれる。

```
www.google.com -> 216.58.211.206
```
