npm は Nodejs のパッケージを管理するためのソフトウェアです。
Nodejs をインストールした時に付属でついてきます。

### npm なぜすごいのか？

昔からソフトウェア開発で課題になっていたのが dependency hell と呼ばれる問題。
例えば、
複数の dependency に頼っているソフトウェアがあります
dependencyA -> jquery(ver 3.0.0)
dependencyB -> jquery(ver 1.0.0)
この時 jquery のバージョンが異なっていることを加味してそれぞれ適切なバージョンをインストールすることができる

- 衝突をなくすことができる
- 結果的に最低限の役割をもったパッケージを無数にインストールする構造が出来上がる
