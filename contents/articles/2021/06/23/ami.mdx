> A blueprint of a customized EC2 instance that can be used to replicate the instance

### How to create one

1 Start the instance
2 Do your customization
3 Stop the instance
4 Build the AMI -> You will get the ARN
5 Launch a new instance by specifying the ami arn you just built

### Use Case

- When you want to replicate over a instance to another region (All instances are scoped by the region)
