### Module identifiers(Specifiers)

Ways to import modules using the ESM syntax

### Relative Specifiers

```js
import module from "./logger.js"
```

### Absolute Specifiers

```js
import module from "file:///opt/nodejs/config.js"

import module from "/config.js" // be careful prefixing with / doesn't work with Nodejs + ESM
```

#### Bare Specifier

```js
import module from "http"
```

Checks the core modules -> node_modules folder

#### Deep Import Specifiers

```js
import module from "fastify/lib/logger.js"
```

Specify a path within the node_modules folder
