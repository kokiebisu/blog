プログラミング言語にはさまざまなデータ型と呼ばれるものが存在しています。String, Array など種類が色々あり、データをいかにして保存するのかを支持することができます。その中でも今回は配列にまつわる
裏技をまとめて紹介できればと思っています。

### 配列に含まれる数字の出現頻度をマッピングする方法

私がよく以下の状況に遭遇します

> 数字が羅列された配列からその数字の頻度をまとめられたななー

例えば。。。

[1, 5, 3, 6, 3, 4, 5]

とあった時に

{ 1: 1, 3: 2, 4: 1, t: 2, 6: 1}

数字の１は一つしか含まれないため、1->1, 逆に 5 の数字は２個あるので 5->2 といった具合です。
このようにまとめる必要がある場面があるかもしれません。

```js
const map = {}
for (let num of nums) {
  map[num] = ~~map[num] + 1
}
```

これにより

```js
const nums = [1, 3, 5, 2, 5] // map = { '1': 1, '2': 1, '3': 1, '5': 2 }
```

となります
