### 配列に含まれる数字の出現頻度をマッピングする方法

```js
const map = {}
for (let num of nums) {
  map[num] = ~~map[num] + 1
}
```

これにより

```js
const nums = [1, 3, 5, 2, 5] // map = { '1': 1, '2': 1, '3': 1, '5': 2 }
```

となります
